# 五十音学习 - Chrome 新标签页插件

一个简约现代的日语五十音学习插件，替换 Chrome 新标签页，让你在每次打开新标签时学习日语。

## 功能特点

- **五十音展示**：显示假名、罗马字和中英文示例
- **每日刷新**：每天自动切换到新的假名
- **手动轮播**：支持左右切换和随机跳转
- **自定义词库**：可以编辑和扩展词库内容
- **简约设计**：现代简洁的视觉风格，支持深色模式
- **键盘快捷键**：左右箭头切换，空格键随机

## 安装方法

### 方法一：开发者模式安装（推荐）

1. 下载并解压插件文件
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择解压后的插件文件夹
6. 安装完成！打开新标签页即可使用

### 方法二：直接拖拽安装

1. 下载 `.zip` 文件并解压
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 将解压后的文件夹拖拽到扩展程序页面

## 使用说明

### 基本操作

- **左右箭头按钮**：切换上一个/下一个假名
- **随机按钮**：随机跳转到任意假名
- **键盘快捷键**：
  - `←` 上一个
  - `→` 下一个
  - `空格` 或 `R` 随机

### 词库编辑

1. 点击右下角的设置图标（齿轮）
2. 开启「使用自定义词库」开关
3. 编辑词条内容：
   - **假名**：需要学习的字符
   - **罗马字**：发音辅助
   - **示例词**：包含该假名的单词
   - **示例罗马字**：示例词的发音
   - **中文/英文**：示例词的含义
4. 点击「保存」应用更改

## 数据结构

词库数据格式：

```javascript
{
  kana: "あ",           // 假名
  romaji: "a",          // 罗马字
  example_jp: "あい",    // 日语示例
  example_romaji: "ai", // 示例罗马字
  example_cn: "爱",     // 中文含义
  example_en: "Love"    // 英文含义
}
```
